# Howe Smart Solutions – Banco de Dados SQL

Este repositório contém todos os scripts SQL desenvolvidos para o projeto de banco de dados da empresa Howe Smart Solutions.

## Conteúdo
- 01_create_tables.sql
- 02_insert_data.sql
- 03_select_queries.sql
- 04_update_queries.sql
- 05_delete_queries.sql

## Execução
Crie o banco:
CREATE DATABASE howe_solutions;
USE howe_solutions;

Depois execute os arquivos na ordem numérica.
